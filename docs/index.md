# Copyedit with AI

Copyedit text from the CLI using AI

## Overview

Copyedit with AI is a Python package that provides a command-line interface for [brief description of what your package does].

## Quick Start

Install copyedit_ai using pip:

```bash
pip install copyedit_ai
```

Then run the CLI:

```bash
copyedit_ai --help
```

## Features

- Modern Python packaging with `uv` support
- CLI interface built with Typer
- Structured logging with Loguru
- Configuration management with Pydantic Settings
- Type checking with ty
- Code quality tools (ruff, pytest)
- Automated testing and CI/CD
