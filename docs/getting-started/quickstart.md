# Quick Start

This guide will help you get started with Copyedit with AI quickly.

## Basic Usage

After [installing](installation.md) copyedit_ai, you can use the CLI:

```bash
copyedit_ai --help
```

This will show you all available commands and options.

## Common Commands

Here are some common commands to get you started:

### Help

Get help for any command:

```bash
copyedit_ai --help
copyedit_ai [command] --help
```

### Version

Check the version:

```bash
copyedit_ai --version
```
